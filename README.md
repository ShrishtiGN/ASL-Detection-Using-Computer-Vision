# ASL Detection Using Computer Vision

## Overview
The ASL Detection project aims to leverage computer vision techniques to recognize American Sign Language (ASL) gestures. This project is designed to help bridge communication between hearing and deaf communities through the use of technology.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ShrishtiGN/ASL-Detection-Using-Computer-Vision.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ASL-Detection-Using-Computer-Vision
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To use the ASL Detection system, run the following command:
```bash
python asl_detection.py
```
You will then be able to input video streams or images for ASL detection.

## Technologies Used
- Python
- OpenCV
- TensorFlow/Keras
- NumPy
- Matplotlib

## How It Works
The project employs a machine learning model trained on ASL gestures. The model processes input images or video streams, detects hands and interprets gestures based on pre-defined data classes.
